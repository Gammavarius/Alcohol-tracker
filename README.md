Трекер дней трезвости

Веб-приложение для отслеживания дней без алкоголя с календарём, статистикой и мотивацией.

Возможности:
Интерактивный календарь с отметкой дней,
Статистика: общее количество дней, текущая и лучшая серии,
История достижений всех серий трезвости,
Автосохранение в localStorage,
Адаптивный дизайн.

Технологии
Pure JavaScript (ES6+),
HTML5 / CSS3,
LocalStorage для хранения данных,
GitHub Pages для хостинга.

Планы развития
Тёмная/светлая тема,
Второй режим трекера (отслеживание дней с алкоголем).
